# AJToastHub
1. 基于UIWindow的Toast和加载Hub。
2. 主要解决键盘遮挡Toast的问题。
3. 支持消息队列显示

### 支持3种位置显示
```objective-c
typedef NS_ENUM(NSInteger, ToastPosition)
{
    ToastPositionBottom,
    ToastPositionCenter,
    ToastPositionTop
};

```
